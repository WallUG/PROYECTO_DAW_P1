# PROYECTO DAW P1

## DESARROLLO DE APLICACIONES WEB

### Sitio Web Interactivo (HTML + CSS + JavaScript) 

Desarrollar un sitio web informativo e interactivo sobre un tema de libre elección, 
aplicando los principios fundamentales del desarrollo web, integrando estructuras 
semánticas con HTML5, estilos con CSS3, e interactividad mediante JavaScript. El 
sitio debe funcionar bajo la arquitectura cliente y reflejar conocimientos de las 
unidades 1, 2 y 3.
